I'm a lecturer in linguistics and quantitative methods at the University of Manchester. My research has focused on using experimental methods to investigate how people interpret referential forms in real-time. Recently, I started a collaborative experimental project with Sarah Ouwayda at the University of Geneva, experimentally looking at interpretations of mass- and count- nouns in English.

      Apart from my experimental work, I have a keen interest in all major aspects of data science. I regularly use the language to develop functions and routines to process, visualize, and analyze data (see Github repository). Additionally, I have used C++ to implement a number of computationally intensive statistical procedures that were originally written in R (see Github repository for raw scripts). I created an R package that allows users to conveniently call the C++ coded procedures in R (see Github repository for the installable R package). Recently, I coded a collection of robust statistical methods (originally written in R) in the statistical and scientific computing language Julia (see Github repository). Most of my scripts and functions can be found on my github account.
